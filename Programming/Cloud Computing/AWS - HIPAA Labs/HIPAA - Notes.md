HIPAA = Health Insurance Portability and Accountability Act

Two rules

- Privacy Rule -> governs who can see PHI (protected health information) and under what circumstances.
	- Would shape who gets what IAM role

- Security Rule -> governs technical and administrative safeguards for electronic PHI (ePHI). Three categories -> admin, physical and technical

## Security Rule Safeguards

Admin -> policies and training 

Physical -> Data center access

Technical -> Encryption, access control, audit logging, integrity 

## Security Rule Codes
Each section has a number like -> §164.312(a)(2)(iv)
- §164.308 = administrative safeguards
- §164.310 = physical safeguards
- §164.312 = technical safeguards (your section)
- §164.314 = organizational requirements (BAAs, contracts)
