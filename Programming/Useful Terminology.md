
#### This is a note to store useful programming vocabulary

--- 
[[Encryption]]
Encrypted at Rest -> Encryption on data that is not moving. If you look at the data that is encrypted at rest you will see cypher text, once you start an ETL the encryption is removed. 

[[Cryptocurrency]]
Hash Rate -> How fast a computer can solve a hash. A hash is a complex mathematical puzzle. 

Collation -> Rules to determine how to compare and sort character data based on specific criteria 

Fail Over -> Automated or manual process where a failed or degraded primary system switches to a secondary back up system with minimal disruption to services. 

SKU -> Used to track and manage different versions, editions, configurations of software products. 
- When you deploy software you need to keep track of all the licenses and versions. This is why we use SKU unique identifier to track each version.

[[Programming]]
Input Binding -> Point a function to a file to use as input.
Out Binding -> Creates an output location for the function.
